## 📚 知识库

### 01. 考勤制度

#### 多系统并行使用

| 系统 | 用途 | 说明 |
|-----|------|------|
| **elink** | 每日上下班打卡 | **唯一结算依据** |
| **刷脸机** | 每日上下班刷脸 | 后续会纳入考勤结算依据 |
| **钉钉** | 每日上下班打卡 | 不纳入结算，作为南网辅助出勤认定 |
| **天阳APP** | 每日上下班打卡 | 内部记录 |

**重要**：每天需完成两次打卡（上班、下班）并及时填报工时和报工，缺卡需先补卡再报工。

#### 打卡要求

**时间规范**：
- 系统默认显示上下班时间为 9:00-17:00（但非实际应出勤时间）
- 需按照项目组要求打卡，确保考勤数据完整
- 迟到早退（即使1分钟）属于违规行为

**定位规范**：
- 严禁在路上打卡或定位异常不报备
- 如有外出需求，请先前往 elink 操作外出申请

**拍照留底**：
- 每次刷脸门禁（最早上班一次、最晚下班一次）立即拍照
- 确保照片清晰显示时间
- 若南网关导出的考勤表出现异常，可提供拍照图片作为申诉证据

#### 工时填报

**elink 工时填报指引**：
1. 工时日志分为上午与下午两栏，需每日打完下班卡后填写
2. 工作日志所记录的工作内容及工作量应可支撑当天的工作量，尽量描写详细和量化（15-100字）
3. 工时日志所记录内容不可连续一周都一模一样，长期事项应体现量化指标
4. 使用"配合""协助"等字眼替代"负责"等字眼
5. 规避采购、立项、可研、党建、招聘等敏感类信息
6. 请假、休假不能填写工时日志（什么都不要写）
7. 当周正常工时支持在周内反复修改，截止周五晚上12点

**异常工时处理**：

| 情况 | 处理方式 |
|-----|---------|
| 忘记打卡 | 上传补卡审批通过的记录 + 正常办公佐证材料 |
| 请假半天 | 无需佐证，在产生原因备注上午/下午请假 |
| 迟到/早退 | 半天工时不予审批 |
| 工时未填报 | 当天不予结算 |

**系统锁定时间**：
- **elink工时**：次月15号锁定上月工时
- **elink补卡**：次月10号锁定上月补卡系统
- **天阳APP补卡**：缺卡需在3个工作日内补卡
- **每周工时**：周五晚22点锁定

#### 补卡规范

**补卡次数限制**：
- **elink**：每月补卡次数限制为6次，截止时间为次月10日
- **天阳APP**：暂未设置月补卡上限，但缺卡需要在3个工作日内（含当天）发起，超时只能走申诉流程（一年3次机会）

**补卡注意事项**：
- 补卡时间需按实际出勤时间填写（不要用系统默认时间）
- 补卡上班时间请选择8:30或者8:30之前
- 补卡下班时间请选择17:30之后
- 工作时长不满足8小时的将按4小时半天结算
- 补卡证明人不能选自己，否则审批驳回

**补卡证据材料**：
- 账号未给：需提供门禁照片（需含日期时间及人脸）或钉钉打卡记录
- 平时忘打卡：可附带同事间elink上讨论工作截图或钉钉打卡记录
- 加班不能补卡

**补卡异常处理**（定位异常、系统BUG、门禁失灵）：
- 10分钟内完成：钉钉打卡 + 天阳打卡 + 门禁拍照 + elink工作沟通截图
- 四证打包发天阳项目助理备案
- 命名格式：姓名+日期+异常类型（如"张三0529定位漂移"）

#### 请假流程

**请假三步走**：
1. **取得南网项目领导同意** - 这是请假流程的第一步
2. **在elink发起请假申请** - 详细填写请假原因、时间等信息
3. **在天阳APP同步发起请假申请** - 入口：工作台 → 考勤休假 → 申请休假

**注意事项**：
- 请假流程需提前申请，不能后补
- 请假需天阳负责人进行审批
- 请假流程成功后，不能撤回及取消
- 请假当天无需填报工时
- 请假须同步elink或邮件报备给南网部门项目经理并取得同意
- 请假最小时间单位为0.5天
- 如上午请假下午回到场地需在13:30前进行elink以及门禁刷脸或指纹打卡
- 三天以上的假期需elink或邮件同步至南网部门总负责

**半天请假**：
- 需在elink与天阳APP完成上下班打卡
- 如下午请假，中午离岗需要打下班卡
- 忘打卡或系统打不上要补卡

**销假**：
- 请假流程完成后，如需撤回、取消或提前结束假期，提交销假完成审批即可
- 须同步线下报备给项目经理并取得同意

#### 外出

- 发起流程，选择申请类型为外出，填报单据内容，提交
- 外出打卡地点无限制，请按各单位要求打卡

#### 加班规范

**决定南网结算单加班数据的两步流程（缺一不可）**：
1. 南网部门在加班开始时间之前做加班申请且审批完成
2. 员工在elink按实际时间打卡（补卡无效）

**加班计算**：
- 工作日19点开始计算加班，一小时为单位
- 加班申请默认扣除上午休息时间12:00-13:30、下午休息时间17:30-19:00
- 加班最大时长取南网各部门设置的最大加班时长
- 加班必须有上下班打卡时间，补卡无效
- 加班无需单独填报日志

**加班注意事项**：
1. 加班由南网部门项目经理发起，申请单据流程完成时间应早于加班开始时间
2. 加班时间以小时为单位，最低加班时长为1小时
3. 工作日上午忘打卡需在加班开始前完成补卡申请及审批
4. 工作日上午请假中午需进行elink以及门禁刷脸或指纹打卡，当晚加班才能计算到加班时长
5. 周末及节假日加班申请需在加班开始前的前一关结束流程
6. 法定节假日加班，除系统申请外，需同步以邮件/elink形式请示部门负责人并取得同意

**加班异常处理**：
- 如加班数据异常，优先检查上述两步流程是否完成
- 若流程无误仍无数据，请联系南网部门助理（非天阳人员）核查
- 天阳仅以南网最终结算单为准申请加班费

#### 出差流程

**出差前**：
1. 联系天阳项目助理，提前报备咨询相关流程
2. 南网发出出差申请（elink或者邮件均可）
3. 出差确认表扫描件（注意签章完整）

**出差中**：
1. 住宿费证明材料（含发票和入住清单）- 开票头自行查询：天阳科技app → 我的 → 开票信息
2. 城际交通费证明材料（电子版，不要打印纸质的）

**注意**：涉及到酒店住宿、城际交通费用，均需自费，并开具好相关发票、行程单等

**出差后**：
1. 联系天阳项目助理，上交出差申请、确认表、发票、行程单等资料
2. 前往天阳OA报销
3. 报销学习入口：天阳APP → 工作台 → 天阳学堂 → 知识库 → 搜索"财务报销&贴票流程"

#### 薪资结算

- **发薪时间**：每月15号
- **发薪依据**：天阳OA报工数据（不包含当月加班费）
- **后期核对**：公司收到南网结算单后，会对员工当月考勤数据再次核对
- **多退少补**：如有加班会在收到结算单当月申请，南网如有考勤异常扣款也会进行补扣
- **注意**：考勤记录影响长期结算，非一次性完结；异常考勤可能导致数月后薪资调整

### 02. 行为规范与信息安全

#### 保密制度

**核心责任**：
1. 承担本人职责范围内的信息安全责任
2. 确保接触的各类信息具备保密性、完整性、可用性

**信息操作规范**：
1. 不违规记录、存储、复制、传递秘密信息
2. 不擅自将内部系统（含南网标识或内部数据）向互联网开放
3. 测试数据需脱敏处理，严禁使用真实数据（除非经授权）

**内容发布与言论纪律**：
1. 不发布或泄露公司商业秘密、经营信息
2. 不散布可能对公司产生负面影响的资讯或言论
3. 未经授权，不对外发布任何公司相关信息

**应急与报告机制**：
1. 发生或可能发生信息安全事件时立即上报
2. 按要求采取应急处置措施，保留原始记录
3. 配合调查并落实整改要求

#### 网络制度

**终端与软件安全管理**：
1. 严禁空口令、弱口令
2. 严禁安装未经授权、来源不明的软件（如QQ、微博、抖音等）
3. 不得关闭或卸载防病毒、防泄露等安全管控软件
4. 严格遵守公司终端管理及网络准入规范

**账号密码与信息保护**：
1. 严禁空口令、弱口令
2. 禁止在暴露区域张贴明文密码或完整IP地址
3. 离开工位必须锁屏，重要文件加密保存
4. 浏览器不保存账号密码，定期清理浏览记录

**网络行为与内容管控**：
1. 严禁将公司信息发布至互联网公共平台
2. 严禁私自搭建无线热点
3. 不得绕过或破坏各类网络安全管控措施
4. 严格遵守公司各项网络安全规定

**应急响应与规范遵守**：
1. 发现疑似网络安全事件立即上报，保留现场
2. 积极配合网络安全事件调查
3. 严格遵守公司各项网络安全规定

**重要**：严禁"一机双网"

#### elink四大操作规范

**文件传输需加密**：
1. 重要文件必须加密后通过elink传输
2. 超过500MB的附件，应使用经技术部认证的硬盘进行拷贝传输

**内容合规要求**：
- 禁止在elink群发表负面、攻击、政治敏感言论
- 一旦被截图投诉，视为严重违纪

**账户登录规范**：
- elink禁止勾选自动登录
- 离开工位须退出，防止他人未经授权访问账户

**数据留存管理**：
1. 后台保留数据不得超过3个月
2. 定期清理历史数据与缓存文件

#### 服务行为规范

**工牌与办公纪律**：
1. 严禁伪造、涂改、盗用或违规借用工牌
2. 入场后全程规范佩戴
3. 凭工牌/临时工牌进出场，不尾随他人通行
4. 遵守场所安全管理规定
5. 严格按照服务时间段提供服务，不迟到、不早退

**着装与仪容要求**：
1. 着装整洁得体，不穿背心、无袖衫、超短裙、夸张服饰等
2. 不染夸张发色，不留怪异发型，不佩戴夸张头饰
3. 办公场所禁止穿拖鞋、凉鞋、洞洞鞋（恶劣天气除外）

**办公行为规范**：
1. 在指定办公区域工作，不随意串岗、串楼，擅离职守
2. 办公时间不从事与工作无关的活动（如游戏、购物、看小说等）
3. 离开工位时关闭电脑

**环境与安全意识**：
1. 主动保持办公区域卫生整洁，物品摆放有序
2. 办公区域内严禁吸烟，自觉维护环境整洁
3. 最后离开者负责关灯、关窗

### 03. 常见问题与违规案例

#### 高频违规Top5场景

**考勤管理违规**：
1. **【忘打卡、忘填工时】** 导致考勤数据不完整，直接影响薪资与项目结算
2. **【定位飘移不报备】** 未及时报备会造成考勤数据异常，引发审核风险
3. **【外出不申请】** 在非常驻地打卡需提前申请，否则影响结算流程与数据有效性

**信息安全违规**：
1. **【安装非授权软件】** 违反安全规定，可能导致数据泄露、系统故障甚至病毒入侵
2. **【泄露屏幕信息】** 任何形式的敏感信息泄露都将面临严肃处理，包括法律追责

#### 违规成本计算器

| 类型 | 后果 |
|-----|------|
| **考勤违规** | 按日薪扣款（迟到早退、缺卡等） |
| **安全违规** | 按合同罚款（安装非授权软件、一机双网等） |
| **形象违规** | 按项目扣分（着装不规范、办公纪律差等） |
