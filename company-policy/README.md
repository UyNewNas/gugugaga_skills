# 📋 公司制度查询 - OpenClaw 技能

随时查询公司考勤、行为规范、信息安全等制度政策。

## ✨ 核心功能

- **⏰ 考勤管理** - 打卡、工时、请假、加班、出差流程
- **👔 行为规范** - 服务行为、着装要求、办公纪律
- **🔒 信息安全** - 保密制度、网络安全、elink使用规范
- **⚠️ 违规案例** - 常见违规场景及后果
- **📞 联系方式** - 项目联系人及支持渠道

## 🚀 快速开始

### 安装

```powershell
# 复制到 OpenClaw 技能目录
xcopy /E /I company-policy %USERPROFILE%\.openclaw\skills\company-policy

# 重启 Gateway
openclaw restart
```

## 💬 使用示例

**你问：** "我明天想请假，应该怎么操作？"

**AI 回答：**
```
📋 请假流程

请假需要三步走：

1️⃣ 取得南网项目领导同意
   - 这是请假流程的第一步，确保请假申请符合项目要求

2️⃣ 在 elink 发起请假申请
   - 详细填写请假原因、时间等信息
   - 确保申请流程完整

3️⃣ 在天阳 APP 同步发起请假申请
   - 入口：工作台 → 考勤休假 → 申请休假

⚠️ 注意事项：
- 必须提前申请，不能后补
- 请假流程成功后不能撤回及取消
- 请假当天无需填报工时
- 须同步 elink 或邮件报备给南网项目经理
- 请假最小时间单位为 0.5 天
- 三天以上假期需同步至南网部门总负责

💡 半天请假记得在 elink 和天阳 APP 完成上下班打卡！
```
