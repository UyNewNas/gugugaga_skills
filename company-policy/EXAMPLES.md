# 公司制度查询 - 使用示例

## 示例 1：打卡问题

### 用户输入
```
打卡有什么要求？
```

### 期望输出
```
📋 打卡要求

## 多系统并行使用

你需要使用多个系统进行打卡：

| 系统 | 用途 | 重要性 |
|-----|------|-------|
| **elink** | 每日上下班打卡 | ⭐ **唯一结算依据** |
| **刷脸机** | 门禁刷脸 | 后续会纳入考勤结算依据 |
| **钉钉** | 每日上下班打卡 | 仅作为辅助记录 |
| **天阳APP** | 每日上下班打卡 | 内部记录 |

## 时间规范

- 系统默认显示 9:00-17:00（但非实际应出勤时间）
- 需按照项目组要求打卡
- 迟到早退（即使1分钟）属于违规行为

## 定位规范

- ❌ 严禁在路上打卡
- ✅ 定位异常需立即报备
- ✅ 外出需先在 elink 申请

## 拍照留底（重要！）

每次刷脸门禁后立即拍照：
- 最早上班一次
- 最晚下班一次
- 确保照片清晰显示时间

💡 若考勤数据异常，照片可作为申诉证据！
```

## 示例 2：工时填报

### 用户输入
```
工时怎么填？
```

### 期望输出
```
📋 工时填报指南

## 基本要求

1. **填报时间**：每天打完下班卡后填写
2. **格式**：分为上午和下午两栏
3. **字数**：15-100字
4. **内容**：详细且量化

## 内容规范

✅ **应该**：
- 具体描述工作内容，支撑当天工作量
- 使用"配合""协助"等字眼
- 长期事项应体现量化指标

❌ **不应该**：
- 连续一周内容一模一样
- 包含敏感信息（采购、立项、可研、党建、招聘）
- 请假当天填写工时（留空即可）

## 注意事项

| 情况 | 处理方式 |
|-----|---------|
| 请假/休假 | 不填工时日志（留空） |
| 忘记打卡 | 上传补卡审批 + 工作佐证 |
| 请假半天 | 备注上午/下午请假 |

## 截止时间

- **每周**：周五 22:00（周内可修改）
- **每月**：次月 15 日（锁定）

⚠️ 工时未填报 = 当天不予结算！
```

## 示例 3：请假流程

### 用户输入
```
我明天想请假，应该怎么操作？
```

### 期望输出
```
📋 请假流程

## 请假三步走

### 第一步：取得南网项目领导同意
- 联系南网项目领导
- 取得口头或书面同意
- 这是请假的前提条件

### 第二步：在 elink 发起请假申请
- 登录 elink
- 填写请假申请
- 包含：原因、时间、时长
- 提交审批

### 第三步：在天阳 APP 同步发起请假申请
- 打开天阳 APP
- 路径：工作台 → 考勤休假 → 申请休假
- 填写与 elink 相同的信息
- 提交审批

## 注意事项

⚠️ **必须提前申请** - 不能后补！

⚠️ **审批后** - 不能撤回及取消

⚠️ **请假当天** - 无需填报工时

⚠️ **同步报备** - 须 elink 或邮件报备南网项目经理

## 半天请假特别说明

如果请半天假：
- ✅ 必须在 elink 和天阳 APP 完成上下班打卡
- ✅ 如下午请假：中午离岗需打下班卡
- ✅ 忘打卡或系统异常：必须补卡

## 三天以上假期

额外要求：
- 需 elink 或邮件同步至南网部门总负责

---

💡 需要帮助？联系助理何紫班（微信或 elink）
```

## 示例 4：加班流程

### 用户输入
```
加班费怎么算？
```

### 期望输出
```
📋 加班流程

## 决定加班数据的两步流程（缺一不可）

### 第一步：加班申请
- **谁发起**：南网部门项目经理
- **何时发起**：加班开始时间之前
- **要求**：审批必须在加班开始前完成

### 第二步：打卡
- **在哪里**：elink
- **何时**：按实际时间打卡
- **注意**：补卡对加班无效！

## 加班计算

| 类型 | 开始时间 | 说明 |
|-----|---------|------|
| 工作日 | 19:00 | 最小单位1小时 |
| 周末 | 按申请 | 申请需在加班前完成 |
| 法定节假日 | 按申请 | + 邮件/elink 请示部门负责人 |

## 扣除时间

- 12:00-13:30（午休）
- 17:30-19:00（晚餐）

## 重要规则

✅ 必须有上下班打卡时间
✅ 加班无需单独填报日志
❌ 补卡对加班无效
❌ 不要提前打卡下班（加班无效）

## 上午请假 + 晚上加班

如果上午请假，晚上想加班算时长：
- 中午必须在 elink 和门禁刷脸打卡（13:30前）

## 加班数据异常？

1. 检查两步流程是否都完成
2. 联系南网部门助理（不是天阳人员）
3. 在最终结算单发出前解决

---

💡 加班费以南网最终结算单为准
```

## 示例 5：补卡流程

### 用户输入
```
忘记打卡了怎么办？
```

### 期望输出
```
📋 补卡流程

## 补卡次数限制

| 系统 | 限制 | 截止时间 |
|-----|------|---------|
| **elink** | 6次/月 | 次月10日 |
| **天阳APP** | 无月限制 | 3个工作日内 |

⚠️ 超过限制 → 走申诉流程（一年仅3次机会！）

## 补卡步骤

### 1. 填写实际时间
- 上班补卡：8:30 或之前
- 下班补卡：17:30 之后
- ❌ 不要用系统默认时间

### 2. 选择证明人
- 选择同组同事
- ❌ 不能选自己

### 3. 上传证据

| 情况 | 所需证据 |
|-----|---------|
| 账号未给 | 门禁照片（含日期时间人脸）或钉钉记录 |
| 平时忘打卡 | elink工作讨论截图或钉钉记录 |

### 4. 提交审批
- 审批人：天阳项目经理

## 异常处理（定位漂移、系统BUG、门禁失灵）

⏰ **10分钟内**，收集四证：
1. 钉钉打卡
2. 天阳APP打卡
3. 门禁拍照
4. elink工作沟通截图

📤 四证打包发给天阳项目助理备案

📝 命名格式：姓名+日期+异常类型
示例："张三0529定位漂移"

---

💡 工作时长必须满8小时，否则按半天结算！
```

## 示例 6：信息安全

### 用户输入
```
网络安全有哪些规定？
```

### 期望输出
```
📋 网络安全规定

## 终端与软件安全

✅ **应该**：
- 使用强密码
- 遵守公司终端管理规范

❌ **不应该**：
- 使用空口令、弱口令
- 安装未经授权的软件（QQ、微博、抖音等）
- 关闭或卸载杀毒软件

## 账号密码保护

✅ **应该**：
- 离开工位锁屏
- 重要文件加密
- 定期清理浏览器记录

❌ **不应该**：
- 在暴露区域张贴密码或IP
- 浏览器保存密码

## 网络行为

❌ **严禁**：
- 将公司信息发布到互联网
- 私自搭建无线热点
- 绕过安全管控措施
- "一机双网"

## elink 使用

✅ **文件传输**：
- 重要文件加密
- 超过500MB用认证硬盘

❌ **内容**：
- 禁止负面、攻击、政治敏感言论
- 截图投诉 = 严重违纪

❌ **登录**：
- 禁止自动登录
- 离开工位须退出

## 应急响应

🚨 发现疑似网络安全事件：
1. 立即上报
2. 保留现场
3. 配合调查

---

⚠️ 违规可能导致合同罚款和法律追责！
```

## 示例 8：出差流程

### 用户输入
```
出差需要准备什么？
```

### 期望输出
```
📋 出差流程

## 出差前

1. **联系天阳项目助理** - 提前报备咨询流程
2. **南网发出出差申请** - elink 或邮件均可
3. **出差确认表扫描件** - 注意签章完整

## 出差中

1. **住宿费证明材料**
   - 发票 + 入住清单
   - 开票头查询：天阳APP → 我的 → 开票信息

2. **城际交通费证明材料**
   - 电子版（不要打印纸质）

⚠️ **注意**：酒店住宿、城际交通费用均需自费，需开具好相关发票、行程单

## 出差后

1. **提交资料** - 联系天阳项目助理，上交出差申请、确认表、发票、行程单等
2. **天阳OA报销**
3. **学习入口** - 天阳APP → 工作台 → 天阳学堂 → 知识库 → 搜索"财务报销&贴票流程"

---

💡 出差前务必提前报备，确保资料齐全！
```
