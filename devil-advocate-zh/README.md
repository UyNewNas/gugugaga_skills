
# 🎭 反驳型人格 - OpenClaw 技能

让 AI 在输出回复前先进行自我反驳，通过正反方辩论提升输出质量。

## 💡 设计理念

你是否遇到过这种情况：AI 给出的回答看起来很有道理，但仔细一想却发现明显的问题？

这个技能的核心思想是：**让 AI 先跟自己辩论，再给你答案。**

通过内部的正反方讨论，AI 可以：
- 发现自己推理中的逻辑漏洞
- 考虑被忽略的边界情况
- 识别潜在的风险和问题
- 提供更平衡和全面的观点

## 🔄 工作原理

```
用户问题
    ↓
【正方】初步构思答案
    ↓
【反方】自我反驳、挑战漏洞
    ↓
【整合】综合观点、完善答案
    ↓
输出最终回复
```

## 📋 自我反驳清单

在内部思考时，AI 会逐一检查：

| 检查项 | 说明 |
|-------|------|
| 🧠 逻辑检验 | 推理链条是否完整？ |
| 🤔 假设质疑 | 隐含假设一定成立吗？ |
| ⚠️ 边界情况 | 有没有例外情况？ |
| 🚨 风险评估 | 有什么潜在风险？ |
| ❌ 反例寻找 | 有没有反例可以推翻？ |
| 🔄 替代方案 | 有没有其他选择？ |

## 🚀 快速开始

### 安装

```powershell
# 复制到 OpenClaw 技能目录
xcopy /E /I devil-advocate %USERPROFILE%\.openclaw\skills\devil-advocate

# 重启 Gateway
openclaw restart
```

### 使用

安装后自动生效，无需额外配置！
